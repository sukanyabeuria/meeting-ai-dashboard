# 🤖 Meeting AI Dashboard

A modern AI-powered Meeting Analytics Dashboard built using React, Vite, Tailwind CSS, Recharts, and Framer Motion.

## 📌 Overview

Meeting AI Dashboard is a responsive analytics dashboard that visualizes meeting statistics, platform usage, participant engagement, sentiment analysis, and meeting insights through an elegant dark-themed interface.

## ✨ Features

- Modern dark UI with responsive layout
- Interactive analytics dashboard
- Meeting statistics cards
- Animated bar chart using Recharts
- Sentiment analysis donut chart
- Platform usage visualization
- Time spent in meetings section
- Talk-to-listen ratio analytics
- Responsive sidebar and navigation
- Smooth animations with Framer Motion

## 🛠 Tech Stack

- React.js
- Vite
- Tailwind CSS
- Recharts
- Framer Motion
- React Icons
- JavaScript

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/sukanyabeuria/meeting-ai-dashboard.git
```

Move into the project

```bash
cd meeting-ai-dashboard
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:5173
```
